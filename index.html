🏴 Initial commands:

  • git init | initilize the repository git

  • git add . | adding the change to the ram or staging

  • git commit -m "let a message about the change" | confirm the changes and send to repository

  • git status | offer a description about the status of our archives


❕ Info commands:

  • git log | is a list of commits made

  • git show | let see the history of changes mades our files


🕓 Come back in the time with this commands:

  • git reset [commitID] --soft | return to the specified commit, this method keep the commits in the local repository

  • git reset [commitID] --hard | return to the specified commit , this method delete all commits from [commitID] onwards

  • git checkout [commitID] --hard | allows us to move between olds commits using [commitID]


⥂ Interactions with the remote repository ( GITHUB-GITLAB-ETC... )

  • git clone [repositoryUrl] | get all data from the remote repository

  • git push | send all changes to remote repository

  • git pull | get the actualizations of ultimates changes that were made in the remote repository
